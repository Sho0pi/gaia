**Gaia** answers simple things itself and delegates the rest to **souls** — specialist
subagents it forges once and **reuses** (never recreates). It remembers you across channels
with two-tier memory, runs long jobs as **missions**, and reaches you on Telegram, WhatsApp,
or your terminal.

```bash
curl -fsSL https://gaia-agent.com/install.sh | bash
```
